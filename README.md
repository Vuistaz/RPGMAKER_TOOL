# RPGMAKER_TOOL

Herramienta para extraer y manejar archivos RPG Maker (".rpgmvp", ".rpgmvo", ".rpgmvm", ".png_", ".ogg_", ".m4a_", ".rpgmz").  
Permite desencriptar y organizar tus archivos de proyectos RPG Maker de manera segura.

## Características
- Desencripta archivos rápidamente
- Compatible con RPG Maker MV y MZ
- Interfaz simple y confiable

## Descarga
Descarga la última versión desde los [Releases](https://github.com/Vuistaz/RPGMAKER_TOOL/releases).
